Options

Options could be set up as data-attributes, predefined as window.waterfall object or passed right to the constructor ☞

Possible options

colMinWidth: 240px
Minimal column width in pixels that’s used as a basis for calculating number of columns. Only pixel values accepted for now. If value is undefined — it firstly tries to parse css min-width, and if it fails will fall down to default 240px.
defaultContainerWidth: $(window).width()
If container is hidden, it’s preferrable to pass default width to count on.
colClass: undefined
Class to append to each column
autoresize: true
Bind reflow on window resize
defaultClass: 'waterfall'
Class of container which to init as waterfall by default

Item position

You can rigidly set position of specific item through it’s data-attributes.

data-column
Specifies what column to place item in. Accepted values: 1..n — points out exact column; 'left', 'right', 'first', 'last' — points out the boundary column.
data-float
Defines floating of item, i. e. boundary column (the same as data-column). Accepted values: 'left', 'right'.

In this case you can render columns with items before waterfall inits itself, assigning them class wf-column. Or you can set data-col-class="column" to the container and assign proper class to it’s children, then waterfall will recognize columns instead of items.

API

Methods could be invoked by calling the constructor with the name of a method passed:

$('.waterfall').waterfall('API_meth')

Besides, you can get the waterfall object that represents all the API methods:

var wf = $('.container').data('waterfall');

! Be aware that zepto disables access to the waterfall object.

Methods available

waterfall.reflow()
Recounts required number of columns, redistributes items by columns. Optimized for speed, so it takes minimal possible calcs.
waterfall.add(item)
Appends a new item or a bunch of items to layout. Using this is more preferrable than jQuery dom-inserting methods like container.append(item). Waterfall will take care of optimal placing and appending.
waterfall.add(
$('<div class="item">Some item.</div>')
); //one item

waterfall.add(
'<div class="item">Item 1.</div>' +
'<div class="item">Item 2.</div>'
); //few items
